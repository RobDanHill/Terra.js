./Terra.js

Javascript procedural terrain generator

Download as .zip Download as .tar.gz View on GitHub

> WASD to move Q and E to yaw mouse to control pitch

> Terra.js is a simple random terrain generator that uses the diamond-square algorithm to generate a height map.

> Terra.js is rendered using Three.js, but you could render the height map any way you wanted to; pure WebGL, canvas, ascii, pieces of toast, the list could go on and on, use your imagination.
